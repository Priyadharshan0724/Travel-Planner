# Travel Planner

A small front-end project for planning travel. It includes a simple HTML UI, styles, and JavaScript behavior.

## Files

- [index.html](index.html) — Main page / UI
- [app.js](app.js) — JavaScript logic
- [style.css](style.css) — Styles

## Run

- Open [index.html](index.html) in your browser (double-click the file or serve with a static server).

## Notes

- No build step or external dependencies required.
- To deploy, serve the files from any static host or use GitHub Pages.
